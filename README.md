# guitar-chords
a program that will find all posibble way to play a chord  for a given chord on a given stringed instrument  use javafx and java

usage:
download the program here https://github.com/macmanmatty/guitar-chords/releases then enter chord to have all possible way to play it displayed.
The default instrument is a 6 string guitar in standard tuning  but you can change that in settings.  
You also click the sharps or flat  checkbox to have notes displayed as sharps or flat.  IE C# or Db.  
You can create a new instrument  under settings too just click the new instrumet button and enter the number of strings.
Then enter the note and octave for each string and the nubmer of frets and the start fret for odd instruments like a banjo.
